nengo_mpi
=========

An MPI backend for the [nengo](https://github.com/nengo/nengo) neural simulation library. Docs can be found [here](https://nengo-mpi.readthedocs.org/).

Tested with python 2.7 and 3.5, and with nengo 2.1.
